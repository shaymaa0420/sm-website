This is my first personal website! It has a pink and teal theme, and features 3 pages: a Home page, where you can briefly read what the website is, an About Me page, where you can read about me (my hobbies and interests!), and a Projects page, where you can get an insight into a previous coding project I've made. Each page has the same theme, but a different picture on the right!

To get to these pages, you can use the menu on the left side of the screen. To get to the Home page, click "Home", to get to the About Me page, click "About Me", and to get to the Projects page, click "Projects". 

To make this website, I used HTML and CSS.
